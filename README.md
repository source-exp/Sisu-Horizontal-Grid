# Sisu UI Grid Fix

A custom Stylus CSS script to improve the layout of `sisu.helsinki.fi`. It forces the grid into three columns and transforms the period list into a horizontally scrollable row.

## Features
- **Horizontal Flow:** Converts vertical period lists into a clean horizontal view.
- **UI Cleanup:** Removes unnecessary vertical lines and decorative markers for a flatter look.
- **Dark Reader:** Also optimized!

## Installation Instructions

### 1. Install Stylus
If you haven't already, install the Stylus browser extension:
- [Stylus for Chrome/Edge](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
- [Stylus for Firefox](https://addons.mozilla.org/en-US/firefox/addon/styl-us/)

### 2. Add the Code
1. Click on the Stylus icon in your browser toolbar.
2. Click **Manage**.
3. Click **Write new style**.
4. Give it a name (e.g., "Sisu Grid Fix").
5. Copy and paste the code from `sisu-grid.user.css` in this repository into the editor.
6. Click **Save** on the left-hand sidebar.

### 3. Usage
Once saved, the style will automatically apply whenever you visit `sisu.helsinki.fi`.
